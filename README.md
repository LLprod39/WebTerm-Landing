# WebTerm Landing

Отдельная статическая лендинг-страница для [WebTerm](https://github.com/LLprod39/WebTerm).

Не является частью приложения. Код продукта (`WebTrerm` / репозиторий) здесь не меняется.

## Просмотр

Из этой папки:

```bash
# Node
npx --yes serve .

# или Python
python -m http.server 5500
```

Откройте `http://127.0.0.1:3000` (serve) или `http://127.0.0.1:5500`.

Можно открыть и `index.html` напрямую, но копирование в буфер надёжнее через http(s).

## Состав

| Путь | Назначение |
| --- | --- |
| `index.html` | Разметка и копирайт |
| `css/styles.css` | Стили, макеты UI |
| `js/main.js` | Вкладки, copy, меню, reveal |
| `assets/favicon.svg` | Иконка |

Макеты интерфейса на странице — схематичные CSS-блоки, не скриншоты продукта.

## Тон

Проект на ранней стадии. Без хайпа, с рабочими командами установки и ссылкой на GitHub.
